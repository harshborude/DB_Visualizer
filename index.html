Free SQL to ER Diagram Visualizer

Instantly transform your raw SQL files into interactive Database Schema Diagrams. Understand your architecture, optimize queries, and analyze table relationships for free.

Features

  • Drag and Drop UI: Upload your SQLite, PostgreSQL, or MySQL `.sql` files instantly.
  • Interactive Canvas: Pan, zoom, and explore connections in a fast, responsive React Flow interface.
  • Pathfinder & Query Builder: Find shortest join paths between tables and visually construct optimized SQL queries.
  • Schema Health Check: Automatically analyze your database for 1NF violations, missing keys, and optimization opportunities.

How to use

Simply drop your database export file into the dropzone. The application works entirely in your browser without uploading your sensitive schema to any server.
